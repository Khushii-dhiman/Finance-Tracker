# Finance Tracker (Django Project)

## 📌 About Project
This is a web-based Finance Tracker application developed using Django. It helps users manage their income, expenses, and budgeting efficiently.

## 🚀 Features
- Add and track income & expenses
- Budget management
- User-friendly interface
- Dynamic data handling

## 🛠️ Tech Stack
- Python
- Django
- HTML, CSS, JavaScript
- SQLite

## ▶️ How to Run
1. Clone the repository  
2. Install dependencies:
   pip install -r requirements.txt  
3. Run server:
   python manage.py runserver
